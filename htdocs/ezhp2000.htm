What are Parler.ca Posts?

Posts are full html web pages created by inputting variables into an online form. The image below shows some of the most common options:

Creating a post

Posts are created by completing a simple form. Here's a rundown of the form fields and what they do:

YOUR NAME
Self explanatory - your name. This is used as a link to your Email address on your post. This is a required field as it also provides necessary Parler.ca account information.

LOGIN ID
This should be a single word that will be used to log into your account to either edit or delete your home page - it shouldn't be easy to guess! This is a required field.

COUNTER OPTIONS ( must be turned on by administrator):
Select the Do Not Use Counter radio button for no counter
 Select the Use White Numbers On Black Counter radio button for this support.
 Select the Use Black Number On White Counter radio button for this support
 You may also set and reset the amount of hits displayed by entering a number in the Set or Reset Hits Entry Field

FILE NAME
The file name of your post consisting of letters and numbers only. It should be a single word - spaces will be replaced with underscores. Don't include the file extension. Your URL will become http://parlers.ca/yourname.html. This is a required field.

HEADLINE
The headline of your Post. This appears at the top of your post, as the title of your page, and as your listing on our index page. This is a required field.

SUB HEADING
This appears underneath the heading of your post.

DO YOU WANT THE HEADLINE AND SUB HEADING CENTERED?
Select whether or not you want the heading and sub heading centered on the post. If you select 'No' they will be left justified.

TEXT FONT FACE
The font face of the text on your post. Enter the name of the font you wish to use, followed by some alternatives for computers that don't have your first choice installed. Separate each with a comma. This field defaults to 'verdana, helvetica, arial'.

THUMBNAIL GRAPHIC
Choose a graphic to appear on the index page, or select 'No graphic'. if external images are allowed (another administrator option) then the thumbnail can be anywhere but the full path (URL) must be entered

PAGE GRAPHIC
Choose a graphic to appear beneath your sub heading, or select 'No graphic'. if external images are allowed (another administrator option) then the page graphic can be anywhere but the full path (URL) must be entered

DO YOU WANT THE IMAGE CENTERED?
Select whether or not you want the image centered. If you select 'No' the image will be left justified.

OPTIONAL EXTRA GRAPHICS (option must be turned on)
The extra graphics are not picked but entered manually there can be a maximum of four thumbnails each linking to a full image. Thumnails can also be used with no links. A neat trick is to not use a stock graphic as the page graphic and then enter the location of a full graphic as the first thumbnail which would become the page graphic. The thumbnails are positioned four across centered and below the page graphic. if external images are allowed (another administrator option) is turned on then the graphics and thumbnails can be anywhere but full paths (URL's) must be entered otherwise the name of the graphics only are entered which the script will display from preset directories.

BODY
Input the body text of your page. Press <return> to start a new line, press it twice to start a new paragraph.

TABLE
A table can be used to display financial information, dates, etc. Type the information you want to appear in the first cell. To start a new cell type two colons (::) followed by the contents of the new cell. To start a new row start a new line by pressing <return>. For example: 

this is row 1 cell 1::this is row 1 cell 2::this is row 1 cell 3
this is row 2 cell 1::this is row 2 cell 2::this is row 2 cell 3 etc...

LINKS (option must be turned on)
You can link to three URLs from your post. Enter the URL of each link followed by the name of the link.

TEXT COLOR
Choose the color you would like text to appear as.

LINK COLOR
Choose the color you would like links to appear as.

VISITED LINK COLOR
Choose the color you would like visited links to appear as.

BACKGROUND
Select the background appearance of your post. Selecting 'No background' will set the background appearance of the post to 'browser default' (usually white or gray).

FIRST HORIZONTAL LINE
Choose the appearance of the horizontal line that sits between the splash image and the body text, or select 'No first line'.

SECOND HORIZONTAL LINE
Choose the appearance of the horizontal line that sits between your links and email address, or select 'No second line'.

BULLET
Bullets are small images that are used to define paragraphs of body text. Select a bullet image to use or select 'No Bullet' if you don't wish to use a bullet image - by selecting this body text will become centered.

DISPLAY CONTACT FORM (option must be turned on)
Choose whether or not you want a Contact Me button with a form mailer for your post.

DISPLAY EMAIL ADDRESS (option must be turned on)
Choose whether or not you want your email address displayed on your post.

EMAIL OPTIONS (option must be turned on)
Select the Do Not Display Email Link radio button for no email link display.
Select the Display Email Link radio button to activate this support.
Email Image Selection Options (only work when Display Email Link is Selected):
Select the No email image for your name to be displayed as an email link.
Select an image to display with your email address link (your name will be the tooltip).

To generate your post page hit 'Create Page'. The title of your post will also be added to our list of posts. You will be shown a confirmation screen and given the URL of your post page that you may share.

Editing a post

To edit your post you need to log in by entering your login name, email address and file name.

Editing your post is similar to creating it - you are presented with a similar form. However, when editing your post your original values are returned. You can change the values to edit your post, but you can't however edit your page file name or login. You can create a new login for publising a new post. Or delete your post old post. ID.

For a description of each form field see Creating a post.

Deleting a post

To delete your post enter your login details and click on 'Login'.

User Image Upload (option must be turned on)

To Upload An Image enter your login details, select from the dropdown list the destination for the upload file and click on 'Login'. The next screen to come up will be the upload file selection screen - just Enter or Select the file to upload and click on 'Upload!'. If the administrator has decided to allow user uploads then only the administrator and users with existing pages can upload images. You would have to create a page first. Then upload your new images and then you could Edit your existing page to display your uploaded graphics.

Lost Logins

Enter your post file name (case sensitve with no extension or path information) and click on 'Search'. To find your page file name place your mouse over the graphic or text link on the index page and look at your browsers's status line. The file name will be a part of the status.

Error messages

If you receive any error messages using Parlers.ca it's probably because you either forgot to enter certain information on a previous form, or entered it incorrectly. Go back and try again!
